# INHA_DB_RunningPlatform

러닝 여행 플랫폼 팀 프로젝트 저장소입니다.

## 저장소 구조

- `frontend/`: Vite 기반 프론트엔드 애플리케이션
- `backend/`: 추후 추가 예정인 백엔드 애플리케이션

## 프론트엔드 실행 방법

```bash
cd frontend
npm install
npm run dev
```

## 참고

- 프론트엔드 관련 설명 문서는 `frontend/README.md`에 정리되어 있습니다.
- `package-lock.json`은 `frontend/`에서 관리합니다.
